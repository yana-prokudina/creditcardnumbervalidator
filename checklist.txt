Чек-лист для тестирования программного кода CreditCardNumberValidator.

// Позитивный сценарий:

/ Подход

Тестирование основано на разделении кредитных карт по типу платежной системы.
Первая цифра кредитной карты указывает на соответствующий тип платежной системы:

Мир - 2 
American Express – 3
VISA – 4
MasterCard – 5 
Maestro - 3, 5 или 6
JCB International - 3
China UnionPay - 6
УЭК - 7

Для генерации произвольных номеров кредитных карт American Express, Visa, MasterCard, JCB International используется онлайн-сервис https://www.getcreditcardnumbers.com
Произвольные номера кредитных карт Мир, Maestro, China UnionPay, УЭК генерируются вручную.

/ Данные

Для каждого типа кредитной карты должно быть протестировано не меньше 5-и сгенерированных номеров.

/ Критерии

Карта валидна, если результат вывода работы кода: 'Result is OK'
Карта невалидна, если результат вывода работы кода: 'Result is FAIL'

// Негативный сценарий:

/ На ввод подается:

1. Пустая строка
2. Неполный ввод: количество цифр < 16
3. Избыточный ввод: количество цифр > 16
4. Ввод нулей: количество цифр = 16
5. Смешанный ввод цифр и символов: 1234abcd5678defg
6. Ввод латинских символов: abcdabcdabcdacbd
7. Ввод знаков: !@#$!@#$!@#$!@#$
8. Ввод пробелов
